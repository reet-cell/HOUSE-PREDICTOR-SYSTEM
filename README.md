# House Price Prediction System

## Overview

A Machine Learning-based web application that predicts house prices using historical housing data. The project uses data preprocessing, feature engineering, and Linear Regression to generate real-time predictions.

## Features

* Data preprocessing and feature engineering
* Missing value handling and categorical encoding
* Linear Regression model training
* Interactive Streamlit web application
* Real-time house price prediction

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Model Training
5. Model Deployment using Streamlit

## Future Improvements

* Advanced regression models
* Hyperparameter tuning
* Model performance comparison
